Application Météo

Application Météo

Cette page web est une application météo. Ce code définit un objet "weather" qui utilise l'API OpenWeatherMap pour afficher la météo actuelle d'une ville donnée. Le code récupère la position géographique de l'utilisateur et affiche automatiquement la météo de la ville la plus proche en utilisant une API tiers. Le code permet également à l'utilisateur de rechercher la météo d'une ville spécifique en saisissant son nom dans une barre de recherche. Le code affiche également des indications sur la pollution au niveau géographique de l'utilisateur.
